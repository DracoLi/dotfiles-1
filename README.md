### Install oh-my-zsh
```sh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

```

### Install dotfiles
```sh
cd ~/
git clone git@github.com:DracoLi/dotfiles-1.git
cd ~/.dotfiles
script/bootstrap
script/install
```
